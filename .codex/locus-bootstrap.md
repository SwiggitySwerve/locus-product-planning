# Locus Bootstrap for Codex

This file should be loaded at the start of every session to enable Locus skills.

## Instructions

When this file is loaded, you have access to Locus skills - a comprehensive framework for AI-powered project planning and development.

## How to Use Skills

To use a skill, read its SKILL.md file from the skills directory:

```
Read skills/using-locus/SKILL.md
```

Then follow the instructions in that skill exactly.

## Available Skill Categories

1. **using-locus** - The main skill for project planning (Vision -> Features -> Design -> Build)
2. **01-executive-suite/** - Strategic leadership perspectives
3. **02-product-management/** - Product planning and execution
4. **03-engineering-leadership/** - Technical leadership
5. **04-developer-specializations/** - Domain expertise
6. **05-specialists/** - Specialized skills

## Quick Start

For project planning, start with:
```
Read skills/using-locus/SKILL.md
```

For specific expertise, use skills like:
```
Read skills/02-product-management/product-manager/SKILL.md
Read skills/03-engineering-leadership/tech-lead/SKILL.md
Read skills/04-developer-specializations/core/frontend-developer/SKILL.md
```

## The Rule

If there's even a 1% chance a skill might apply to your task, load and check it.
Skills contain proven workflows and expert techniques - use them.
